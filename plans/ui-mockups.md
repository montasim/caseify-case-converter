### Visual Specifications

**Header:**
- Height: 80px
- Background: Glassmorphism (blur 20px, 70% opacity)
- Logo: Gradient purple-blue "C" in rounded square
- Nav items: Pill-shaped hover states

**Hero Title:**
- Font: 3.5rem (desktop), 2rem (mobile)
- Gradient: Purple-blue to teal
- Animation: Subtle float (Â±2px)

**Converter Card:**
- Background: White with 90% opacity
- Border: 2px gradient (purple-blue â†’ teal)
- Border radius: 24px
- Shadow: Large soft glow
- Textarea: No border, 32px padding, 1.125rem font

**Stats Bar:**
- Background: Muted with 30% opacity
- Badges: Rounded pills with icons
- Hover: Scale 1.05, border highlight

**Action Buttons:**
- Size: 56x56px
- Background: White with subtle shadow
- Hover: Scale 1.1, gradient border
- Icons: Primary color on hover

**Conversion Buttons:**
- Grid: 4 columns (desktop), 2 columns (tablet), 1 column (mobile)
- Height: 56px
- Background: Secondary with 80% opacity
- Hover: Primary gradient, scale 1.05
- Active: Scale 0.95

**Feature Cards:**
- Grid: 3 columns (desktop), 1 column (mobile)
- Background: Glassmorphism
- Icon: Gradient circle background
- Hover: Lift 8px, shadow increase

### Visual Specifications

**Layout:**
- Split: 40% left (info), 60% right (form)
- Gap: 32px between panels
- Both panels: Equal height, glassmorphism

**Info Cards (Left):**
- Icon: Gradient circle (48px), centered icon
- Title: 1.125rem, semibold
- Description: 0.875rem, muted
- Gap: 24px between cards

**Social Links:**
- Pill-shaped buttons
- Background: Muted
- Hover: Primary gradient

**Form (Right):**
- Labels: 0.875rem, uppercase tracking
- Inputs: 16px padding, rounded 16px
- Focus: Gradient border, glow effect
- Submit: Full width, gradient background
- Loading: Spinner animation

**Success State:**
- Icon: 80px circle, green gradient
- Animation: Zoom in + confetti
- Button: Outline style, centered
